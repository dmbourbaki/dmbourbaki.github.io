¿Qué es este gráfico? Muestra |f(x) − Tₙ(x)| — qué tan lejos está el polinomio del valor real. Eje Y en escala logarítmica: más abajo = error más pequeño = mejor aproximación. A medida que n crece, las curvas bajan → el polinomio se acerca más a f(x).
¿Qué es este gráfico? Cada barra es el coeficiente cₖ = f⁽ᵏ⁾(a)/k! del término de grado k. Verde = positivo, rojo = negativo. Si |cₖ| → 0 rápido, la serie converge rápido (p.ej. eˣ: todos positivos crecientes al inicio, luego decrecen).
